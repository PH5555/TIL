# 버블 정렬

'바로 옆에있는 수와 비교해서 바꾼다'

1. 첫번째 요소와 두번째 요소를 비교한다
2. 작은 수가 앞에, 큰 수가 뒤에 오도록 바꾼다
3. 마지막 요소까지 반복하면 제일 큰 수가 맨 뒤에 오게 된다
4. 맨 뒤를 제외하고 1~3과정을 반복한다

```c

#include <stdio.h>

int main(){
	int i, j, temp;
	int num[10] = {5, 4, 9, -8, 4, 50, 12, 45, 11, 10};
	
	for(i = 0; i < 10; i++){
		for(j = 0; j < 9 - i; j++){
			if(num[j] > num[j + 1]){
				temp = num[j];
				num[j] = num[j + 1];
				num[j + 1] = temp;
			}
		}
	}
	
	for(i = 0; i < 10; i++){
		printf("%d ", num[i]);
	}
}

```

# 시간 복잡도

O(N * N)
