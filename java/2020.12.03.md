# 변수

수학에서는 변하는 수이다

하지만 프로그래밍에서는 하나의 값을 저장할 수 있는 메모리 공간!

# 변수의 선언

- 변수를 선언하는 이유

  값(data)을 저장 할 공간을 마련하기 위해서

- 선언 방법 : (변수 타입) (변수 이름); 

```java

int age;

```

# 변수에 값 저장

```java

int age;
age = 20;
int age = 20; //위에 두줄 한번에

```

- 변수의 초기화

  변수에 처음으로 값을 저장하는 것
  
  지역변수는 자동으로 초기화되지 않기 때문에 사용하기 전에 꼭 초기화!
