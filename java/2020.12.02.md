# 글자 출력

- println
출력하고 줄바꿈까지 해준다!

  System.out.println("hello!");

- print
줄바꿈은 안해줘요

  System.out.print("hello!");

- 사칙연산 계산 출력
  System.out.println(3+5);
  System.out.println(3*5);
  System.out.println(3/5);
